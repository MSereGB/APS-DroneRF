# Datos

Los CSV originales de DroneRF no forman parte de este paquete.

`manifests/dronerf_demo_v2_manifest.json` contiene la selección final de 158 archivos y 79
grupos. Cada registro incluye nombre, parte L/H, grupo, partición y frecuencia de muestreo.
La variable `DATA_DIR` debe apuntar a la carpeta donde se guardaron esos archivos.
